## Project Summary

Nova Uni o RP is a GTA RP city resource made for FiveM servers. It is intended to provide roleplay-oriented city content in a packaged form that server owners can slot into a FiveM setup for a city-style RP environment.

The repository is centered on city presentation content rather than a full gameplay system. That makes it a better fit for teams looking to extend a FiveM roleplay server with map or city atmosphere elements connected to GTA RP communities. When updating, take care to keep the resource in step with the rest of the server structure and any custom assets already deployed.

## Installation

1. Download the latest build from the download link above.
2. Put the resource folder into your FiveM server resources directory.
3. Register the resource in your server configuration.
4. Start it alongside the rest of your server assets.

Example server.cfg entry:

start nova-uni-o-rp

If your server folder name is different, make sure the `start` line matches that folder exactly.
